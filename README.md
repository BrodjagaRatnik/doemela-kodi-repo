# Doemela Kodi Repository
Official repository for **Doemela** Kodi add-ons, specifically optimized for **Kodi 21 (Omega)** and **Raspberry Pi 5**.

## 🚀 Featured Add-ons:
* 🕒 Clock Overlay: A lightweight Python-based clock that displays a temporary overlay during media playback.
* <img src="zips/script.clock.overlay/resources/media/screenshot00000.png" alt="Alt text" width="800">
* <img src="zips/service.wireguard.manager/resources/media/icon.png" alt="Alt text" width="30"> WireGuard Manager for NordVPN: A lightweight, high-performance Kodi service addon for LibreELEC 12+ (Kodi 21 Omega). This tool manages WireGuard connections natively via connmanctl, providing a faster and more stable experience than traditional OpenVPN-based addons. For detailed instructions for Add-on WireGuard Manager, please visit our **[Wiki](https://github.com/BrodjagaRatnik/service.wireguard.manager/wiki)**
* <img src="zips/service.wireguard.manager/resources/media/screenshot00002.png" alt="Alt text" width="800">
